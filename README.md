# 🧱 Dimny Texture Packs

## 🛠️ Installation Instructions

1. Download a `.zip` file from the links above.  
2. Move it to your Minecraft resourcepacks folder:  
3. Open Minecraft → Options → Resource Packs → Activate the pack.
